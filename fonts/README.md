This directory contains the local font assets used for deterministic PDF rendering.

Required v1 font files:
- `NotoSerif-Regular.ttf`
- `NotoSerif-Bold.ttf`
- `NotoSansSC-Regular.ttf`

Purpose:
- English and Spanish PDFs use Noto Serif for plain academic readability.
- Mandarin PDF uses Noto Sans SC for Unicode-safe Simplified Chinese rendering.

Notes:
- These files were copied into the repository workspace from fonts already available on the local machine.
- The PDF layout should remain intentionally plain and professional: title, headings, readable margins, and page numbers.
